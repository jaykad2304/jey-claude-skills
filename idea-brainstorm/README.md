# Idea Brainstorm Skill

A Claude skill that transforms any raw idea into a full product strategy analysis.

## What It Does

When you share an idea, Claude instantly analyzes it across 8 dimensions — acting simultaneously as a startup founder, product manager, AI architect, and investor.

## Triggers

Use this skill when you say things like:
- "I have an idea..."
- "Help me brainstorm..."
- "What do you think about building X?"
- "Analyze my idea"

## Output Structure

| Section | What You Get |
|---|---|
| 1. Core Idea Expansion | Improved version, variations, advanced vision |
| 2. Features & Enhancements | Additional features, automation, AI capabilities |
| 3. Use Cases | Target users, real-world scenarios, industry applications |
| 4. Monetization | Revenue models, SaaS angles, upsell paths |
| 5. Competitive Edge | Differentiators, positioning, moat-building |
| 6. Technical Possibilities | Stack, cloud services, automation tools |
| 7. Future Expansion | 6-month, 1-year, and scaling vision |
| 8. Quick Wins | High-impact small features + MVP recommendation |

## Example

**Input:**
> I want to build a Maratha Matrimony website

**Output:** A full 8-section analysis covering community-specific features (gotra/sub-caste filters, Kundali matching, family-gated profiles), monetization via freemium + events + sanghatna partnerships, NRI expansion strategy, and a 6-week MVP plan.

## Installation

Copy `SKILL.md` to your Claude skills directory:

```bash
cp SKILL.md ~/.claude/skills/idea-brainstorm/SKILL.md
```

Then start a new Claude Code session — the skill loads automatically.
